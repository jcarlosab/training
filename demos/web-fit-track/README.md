# Fit Track

App de seguimiento fitness con interfaz drag-and-drop para gestionar rutinas de ejercicio.

![React](https://img.shields.io/badge/react-v18-blue.svg)
![Vite](https://img.shields.io/badge/vite-5-purple.svg)

## Caracteristicas

- Registro de ejercicios y rutinas
- Interfaz drag-and-drop para reorganizar
- Seguimiento de progreso

## Stack

- **React 18** + **Vite**
- **ESLint** - Linting

## Instalacion

```bash
npm install
npm run dev
```
