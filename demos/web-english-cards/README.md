# English Cards

App de tarjetas (flashcards) para practicar vocabulario en ingles.

![React](https://img.shields.io/badge/react-v18-blue.svg)
![Vite](https://img.shields.io/badge/vite-5-purple.svg)

## Caracteristicas

- Tarjetas de vocabulario en ingles
- Modo estudio con repeticion espaciada
- Interfaz sencilla y rapida

## Stack

- **React 18** + **Vite**
- **ESLint** - Linting

## Instalacion

```bash
npm install
npm run dev
```
