# Demos

Coleccion de proyectos de demostracion y experimentos web. Cada subcarpeta es un proyecto independiente con su propio `package.json`.

## Proyectos

| Proyecto | Descripcion | Stack |
|----------|-------------|-------|
| [web-english-cards](web-english-cards/) | App de tarjetas de vocabulario en ingles | React + Vite |
| [web-fit-track](web-fit-track/) | App de seguimiento fitness con drag-and-drop | React + Vite |
| [web-marvel](web-marvel/) | Buscador de personajes de Marvel con infinite scroll | React + CRA |
| [web-task-poker](web-task-poker/) | Planning Poker en tiempo real con WebSockets | React + Vite + Node.js + Socket.io |

## Ejecucion

Cada proyecto es independiente:

```bash
cd nombre-proyecto
npm install
npm run dev
```
